Welcome to Minimal Kiosk Browser (kweb)

This is the default homepage, that will be opened upon start and when you click the "Home" button in the toolbar. You should create your own homepage with your favourite links, often used functions etc., named "homepage.html", and put it in your user's root directory (replacing this one). If you do not know, how to create a HTML page, you can use a WYSIWYG HTML editor like Iceape Composer or the homepagecreator program supplied in the tools archive (check the manual for details).

If you don't want to use a local homepage file at all, you can start Minimal Kiosk Browser from the command line of a terminal like that:

kweb -H http://yourpreferredwebsite

or, if you want to have full keyboard support:
kweb -Hobhrp+-zgtjnediwxyqfc http://yourpreferredwebsite

You can also create a start script like that:

#!/bin/bash
kweb -H http://yourpreferredwebsite

Save it as "startBrowser" (or any name you like) to your Desktop, make it executable and use it to start Minimal Kiosk Browser. Then the browser will always start with your preferred website and return to it, if you click the "Home" button in the toolbar.
